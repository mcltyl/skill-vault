## 🎯 What Are Agent Skills?

Agent Skills are instruction files that teach AI coding assistants how to handle specific tasks. Instead of writing the same prompts over and over, you install a skill once and it just works.

**Compatible with:**
- [Claude Code](https://docs.anthropic.com/en/docs/claude-code)
- [Codex CLI](https://github.com/openai/codex-cli)
- [OpenClaw](https://github.com/openclaw/openclaw)
- Any tool that supports the Agent Skills format

## 🚀 Installation

### Option 1: npx skills (Recommended)

```bash
npx skills add git@github.com:mcltyl/[skill-name].git
```

### Option 2: Manual Clone

```bash
# For Claude Code
git clone https://github.com/mcltyl/[skill-name].git ~/.claude/skills/[skill-name]

# For OpenClaw
git clone https://github.com/mcltyl/[skill-name].git ~/.openclaw/skills/[skill-name]

# For OpenCode
git clone https://github.com/mcltyl/[skill-name].git ~/.opencode/skills/[skill-name]
```
